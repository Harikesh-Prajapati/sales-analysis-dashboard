# 📊 Sales Analytics Dashboard  
### MySQL + Python + Machine Learning + Power BI

An end-to-end Sales Intelligence solution that transforms raw transactional data into actionable business insights using Data Analytics, Machine Learning, and interactive Power BI dashboards.

---

# 🚀 Project Overview

The **Sales Analytics Dashboard** is a complete Business Intelligence project designed to help organizations analyze sales performance, forecast future revenue, identify customer behavior patterns, and support data-driven business decisions.

This project integrates:

- MySQL for database management
- Python for data cleaning & machine learning
- Power BI for interactive dashboards
- Predictive analytics for sales forecasting and churn analysis
